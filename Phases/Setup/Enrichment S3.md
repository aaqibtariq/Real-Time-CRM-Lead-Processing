# Create Enriched S3 Bucket

- Go to:
    - S3 → Create bucket
    - name ->Create Enriched S3 Bucket
    - Block all public access: ON
    - Encryption: SSE-S3
    - Versioning: Optional
    - Create
    - Creater folder target/
